== Description ==

A simple plugin allows you manage your team members.

You can insert your member infomation and display it at front-end by slider or grid.

Shortcode [c4d-team-member-slider count="10"][/c4d-team-member-slider]

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

1. Upload `c4d-team-member` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
